## Specification

This is a source code of the frontend test held by ProSpace. This app is created using Create React App and Redux for state management. I used CSS module that pre-installed with Create React App for styling. All data are stored locally in browser local storage via redux-persist middleware. All the insert and delete actions are served by Redux. There is no backend process occured in this app.

## Getting Started

### `npm install`
Install all the required packages and dependencies.

### `npm start`
Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
