## Specification

This is a source code of the frontend test held by ProSpace. This app using React to build the interface and Redux for state management. All data are stored locally in browser local storage via redux-persist middleware.

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
